# 스택과 큐
## 1. 스택
- 먼저 들어간 값이 가장 나중에 나오는 데이터 구조
- 입출구가 1개
- 1, 2, 3, 4, 5 순서대로 다음 중 스택이 아닌 것은?
    - 1. 3, 4, 5, 2, 1
    - 2. 5, 4, 3, 2, 1
    - 3. 3, 2, 4, 5, 1
    - 4. 4, 2, 3, 1, 5
    - 5. 1, 2, 3, 4, 5

## 2. 큐
- 먼저 들어간 값이 가장 빨리 나오는 데이터 구조
- 입구 1개, 출구 1개

## 3. 깊스너큐
- 깊이 우선 탐색은 스택
- 너비 우선 탐색은 큐
